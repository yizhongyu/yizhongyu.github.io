hexo搭建

博客 hexo
博客
发布日期:   2022-09-01
文章字数:   571

Hexo 是一个快速、简洁且高效的博客框架。Hexo 使用 Markdown（或其他渲染引擎）解析文章，在几秒内，即可利用靓丽的主题生成静态网页。

hexo需要Node.js/Git来支持

  • Node.js(Node.js 版本需不低于 10.13，建议使用 Node.js 12.0 及以上版本)

  • Git

上传node.js/git源码包到服务器并解压

scp C:\Users\zhong\OneDrive\blog\node-v16.17.0-linux-x64.tar.xz root@192.168.200.98:/opt
scp C:\Users\zhong\OneDrive\blog\git-2.37.3.tar.gz root@192.168.200.98:/opt
tar -xJvf /opt/node-v16.17.0-linux-x64.tar.xz -C /usr/local/
mv /usr/local/node-v16.17.0-linux-x64/ /usr/local/node
tar -zxvf /opt/git-2.37.3.tar.gz -C /usr/local/

安装node.js

echo 'export PATH=/usr/local/node/bin:$PATH' >> /root/.bash_profile
source /root/.bash_profile			 #刷新环境变量
node -v & npm -v					#验证

安装git

yum install curl-devel expat-devel gettext-devel openssl-devel zlib-devel gcc-c++ perl-ExtUtils-MakeMaker

编译安装(安装前卸载自带的低版本git)yum remove git

cd /usr/local/git-2.37.3/
./configure --prefix=/usr/local/git
make profix=/usr/local/git
make install

配置git环境变量

echo 'export GIT_HOME=/usr/local/git' >/etc/profile.d/git.sh
echo 'export PATH=$PATH:$GIT_HOME/bin' >> /etc/profile.d/git.sh
source /etc/profile
git --version

npm安装Hexo

npm config set registry https://registry.npm.taobao.org		#添加国内镜像源
npm i hexo-cli -g
hexo init blog
cd blog
npm install
hexo -v			#验证
hexo s			#启动

访问地址 IP端口4000

安装主题

在 hexo 工作文件夹的根目录中执行

npm i hexo-theme-butterfly

编辑_config.yml

theme: butterfly

重载生成博客文件

hexo clean && hexo g

butterfly主题专属

为了减少升级主题后带来的不便,在 hexo 的根目录创建一个文件 _config.butterfly.yml

cp node_modules/hexo-theme-butterfly/_config.yml _config.butterfly.yml

主题配置文件位置：/root/blog/_config.butterfly.yml

插件安装

npm install --save hexo-deployer-git					   #git部署(必选)
npm install --save hexo-admin							  #后台(IP端口4000/admin)
npm install hexo-renderer-pug hexo-renderer-stylus --save	#pug 以及 stylus 的渲染器
npm install hexo-generator-search --save				   #搜索
npm i --save hexo-wordcount								  #文章字数统计

配置调整

hexo配置文件：/root/blog/_config.yml

推送到GitHub

网络不可达时可临时添加代理

export http_proxy='http://192.168.200.29:7890'

生成静态网页并将本地数据部署到设定的GitHub

hexo d
