redis

centos7 redis
集群
发布日期:   2022-05-18
文章字数:   948

Remote Dictionary Server 远程字典服务

安装

yum安装

#yum install epel-release -y		#可选
yum install redis					#安装
systemctl start redis				#启动Redis
systemctl status redis				#查看Redis状态
systemctl stop redis				#停止Redis
systemctl restart redis				#重启Redis
systemctl enable redis				#加入开机自启
firewall-cmd --zone=public --add-port=6379/tcp --permanent		#防火墙开放端口

源码安装

上传redis源码包到服务器

scp E:\app\Linux\redis-5.0.14.tar.gz root@192.168.200.99:/opt

解压

mkdir /usr/local/redis
tar -zxvf /opt/redis-5.0.14.tar.gz -C /usr/local/redis

编译安装

cd /usr/local/redis/redis-5.0.14/
make && make install

将REDIS安装为系统服务并后台启动

cd utils/
./install_server.sh			#此处我全部选择的默认配置即可，有需要可以按需定制

查看REDIS服务启动情况

ps -aux|grep redis
systemctl status redis_6379.service
	启动REDIS客户端并测试
	启动⾃带的redis-cli客户端
redis-cli
set foo bar
get foo
	设置允许远程连接
	编辑redis配置⽂件
vim /etc/redis/6379.conf
	将bind 127.0.0.1修改为0.0.0.0
	然后重启Redis服务
systemctl restart redis_6379.service
	设置访问密码
	编辑redis配置⽂件
vim /etc/redis/6379.conf	
	找到#requirepass foobared
	去掉注释，将foobared修改为⾃⼰想要的密码，保存即可。
	保存，重启Redis服务
systemctl restart redis_6379.service
	auth 密码

使用

redis-server redis.conf				#启动(redis服务加配置文件)
redis-cli -p 6379					#连接(redis客户端加IP加端口号)
redis-cli -h 192.168.200.99 -p 6379	#指定服务器主机名，默认127.0.0.1
config set requirepass "123"		#登录进去后(设置密码123)
config get requirepass				#查看密码
auth 123							#登录

配置文件redis.conf

INCLUDES（包含）

可包含多个配置文件

include /path/to/local.conf

include /path/to/other.conf

NETWORK（网络）

bind 127.0.0.1（默认本地访问）

bind 0.0.0.0（允许远程连接）

protected-mode yes（是否受保护）

port 6379（默认端口）

GENERAL（通用）

daemonize yes（守护进程开启）后台

pidfile /var/run/redis_6379.pid如果是后台运行，需要指定一个pid文件

loglevel notice（debug、verbose、notice、warning）日志级别

logfile /var/log/redis_6379.log（日志文件）

databases 16（数据库数量）

always-show-logo yes（显示logo）

SNAPSHOTTING（快照）

持久化，在规定的时间内，执行了多少次操作，则会持久化到文件.rdb，.aof

save 900 1（900秒内至少有一个key进行了修改，及进行持久化操作）
save 300 10（300秒内至少有10个key进行了修改，及进行持久化操作）
save 60 10000（一分钟内至少有10000个key进行了修改，及进行持久化操作）

stop-writes-on-bgsave-error yes（持久化出错后是否继续工作）

rdbcompression yes（是否压缩.rdb文件）

rdbchecksum yes（校验.rdb文件）

dbfilename dump.rdb（.rdb名称）

dir /var/lib/redis/6379（.rdb文件保存目录）

REPLICATION（主从复制）
SECURITY（安全）

requirepass foobared（去掉前面注释foobared改成密码）

CLIENTS（客户端限制）

maxclients 10000（最大连接客户端一万个）

MEMORY MANAGEMENT（内存管理）

maxmemory （最大内存）

maxmemory-policy noeviction（内存爆了后的策略）

APPEND ONLY MODE（.aof配置）

appendonly no（开关）

appendfilename “appendonly.aof”（持久化文件名）

appendfsync always（每次修改执行同步）

appendfsync everysec（每秒执行一次同步）

appendfsync no（不同步）

appendfilename “appendonly.aof”（.aof名称）

持久化文件

config get dir输出

  1. “dir”
  2. “/var/lib/redis/6379”

将dump.rdb文件放到目录下会自动恢复

集群、哨兵

单台最大内存不应该超过20gb，超过后一主三从

环境配置

只配置从库，不配置主库

info replication（查看当前库的信息）

命令配置从节点SLAVEOF [IP] 6379（暂时），修改从节点配置文件（永久）

replicaof

masterauth
