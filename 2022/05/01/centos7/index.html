centos7基础命令

centos7
os
发布日期:   2022-05-01
文章字数:   2.4k

记录一下使用频次较高的命令，后面再来整理一篇好用的命令

网络

查看网络信息

ifconfig
ip addr
ip address show					#IP地址MAC地址
dhclient						#重新分配IP
ip route show					#路由
ip n
ip -s l
ss								#网络信息
netstat -anp | grep 8080		#查看端口是否已经监听

设置固定IP

vim /etc/sysconfig/network-scripts/ifcfg-[eth0]		#[网卡号]
BOOTPROTO=static									#(none禁止DHCP，static启用静态IP地址，dhcp开启DHCP服务)
IPV6INIT=no											#禁止IPV6
ONBOOT=yes											#开机启动
IPADDR=172.3.4.249									#静态IP地址
NETMASK=255.255.255.0								#子网掩码
PREFIX=24											#掩码长度
GATEWAY=172.3.4.1									#网关
DNS1=218.104.111.114
DNS2=180.76.76.76									#DNS

重启网络服务

systemctl restart network.service			#或
service network restart

其他网络参数

  • 是否允许DHCP获得的DNS覆盖本地的DNSPEERDNS
  • 是否从DHCP服务器获取用于定义接口的默认网关的信息的路由表条目PEERROUTES
  • 唯一标识UUID
  • 如果ipv4配置失败禁用设备IPV4_FAILURE_FATAL=yes
  • 如果ipv6配置失败禁用设备IPV6_FAILURE_FATAL=yes
  • 定义设备名称NAME=“eth 或 ens”
  • address表示广播地址BROADCAST=“address”
  • MAC-address表示指定一个MAC地址MACADDR=“MAC-address”
  • 是否允许非root用户控制该设备USERCTL=yes/no

查看主机信息

hostnamectl
cat /etc/centos-release			#查看系统版本
sestatus						#se状态

修改host

hostname 123				#临时修改
vim /etc/hostname			#永久修改
echo "123 >>/etc/hostname	#或者echo输出到末尾
vim /etc/hosts				#编辑或者echo
echo "IP空格hostname" >>/etc/hosts

用sed修改selinux状态

sed -i "s/SELINUX=enforcing/SELINUX=disabled/g" /etc/selinux/config

其他常用配置文件

  • 配置域名服务客户端的控制文件/etc/host.conf
  • 完成主机名映射为IP地址的功能/etc/hosts
  • 域名服务客户端的配置文件,用于指定域名服务器的位置/etc/resolv.conf
  • 包含了主机最基本的网络信息,用于系统启动/etc/sysconfig/network
  • 系统启动时初始化网络的一些信息以及网卡的配置文件/etc/sysconfig/network-script/
  • 定义了由超级进程xinetd启动的网络服务/etc/xinetd.conf
  • 完成域名与网络地址的映射/etc/networks
  • 设定了主机使用的协议以及各个协议的协议号/etc/protocols
  • 设定主机的不同端口的网络服务/etc/services

防火墙

systemctl status firewalld				#查看防火墙状态
firewall-cmd --list-ports				#查看防火墙开放了哪些端口
systemctl stop firewalld.service		#关闭防火墙
service firewalld start					#开启防火墙
systemctl restart firewalld.service		#重启防火墙
firewall-cmd --zone=public --add-port=80/tcp --permanent		#开启端口
--zone作用域;--add-port=80/tcp添加端口，格式为:端口/通讯协议;--permanent永久生效，没有此参数重启后失效
systemctl stop firewalld.service
systemctl disable firewalld.service								#永久关闭防火墙

使用终端远程登陆

  • ssh空格@IP，cmd进入ssh root@172.3.4.249

    1.tab键			用于自动不全命令/文件名/目录名
2.ctrl+l		清理终端显示
3.clear/cls		清理终端显示
4.ctrl+c		终止当前操作
5.ctrl+r		搜索历史命令
history			显示历史命令

  • scp (secure copy) #linux下基于ssh登录进行安全的远程文件拷贝命令

    • 递归复制整个目录-r

    • 详细方式输出-v

    • 不显示传输进度条-q

    • 允许压缩-c

      scp -r /usr/local/setup root@192.168.1.233:/usr/local/
scp 本地文件	远程用户名@远程ip:远程文件夹/

安装软件

  • 查看vim是否安装rpm -qa|grep vim

  • 在线安装个编辑器yum -y install vim

  • 安装本地rpm包提示安装进度条rpm -ivh xxx.rpm

  • 卸载rpm -e xxx或yum remove -y xxx

  • 强制卸载

    rpm -qa|grep xxx
rpm -e --nodeps xxx

查看软件配置文件路径rpm -qc xxx

vim

  • 使用vi打开文件，默认是命令模式，需要输入a/i进入编辑模式，然后输入文本，按下esc键，回到命令模式
    • 强制保存退出:wq!
    • 写入writew
    • 退出quitq
    • 强制!
    • 保存退出:x
    • 删除光标位置的字符(X删除前一个)x
    • 查找字符串(n下一个N上一个)/xx
    • 复制几行数字YY
    • 剪切几行数字DD
    • 插入P
    • 撤销u
    • 编辑的数据另存为:w [filename]
    • 如果已改动保持并退出，未改动不存储离开ZZ
    • 显示行号(:set nonu相反):set nu

服务管理

查看服务进程

pstree -pu							#进程树
ps -aux|grep xx						#查看所有进程
ps -ef|grep xx						#查看父进程
netstat -tunlp|grep port			#根据端口号查看进程
kill -9 进程数字					 #强制结束进程

操作服务

systemctl start service_name		#服务启动
service service_name start
systemctl stop service_name			#服务停止
service service_name stop
systemctl status service_name		#服务状态
service service_name status
systemctl enable service_name		#加入自启动
chkconfig --add service_name
systemctl disable service_name		#移除自启动
chkconfig --del service_name
开机脚本
vim /etc/init.d/xxx			#新建开机脚本
chmod +x /etc/init.d/xxx	#赋权
chkconfig --add xxx			#加入开机列表
chkconfig xxx on			#状态开
chkconfig --list			#查看开机列表详情
systemctl list-unit-files | grep nginx		#搜索nginx的开机启动状态
systemctl list-unit-files | grep enabled	#搜索全部已启动的服务，enabled 启用；disabled 禁用。

语言

查看系统当前编码vim /etc/locale.conf永久使用中文

LANG="zh_CN.UTF-8"
LC_ALL="zh_CN.UTF-8"
LC_CTYPE="zh_CN.UTF-8"

文件系统

新建

  • 新建文件夹mkdir
  • 递归创建目录，即使上级目录不存在，会按目录层级自动创建目录mkdir -p
  • 新建文本vi vim touch

路径与属性

  • 返回上级目录(cd ../..)上上层cd ..

  • 进入绝对路径目录cd /

  • 进入用户根目录(cd)cd ~

  • 查看当前的路径pwd

  • 显示文件属性(ll)ls -l

    • 查看包括隐藏文件在内的所有文件以及属性ls -al

移动（搬家）命令 > move > mv

  • 要移动的文件 目标地址mv /home/xb.py /tmp/ xb1.py

    • 文件或文件夹改名mv xxx.log xx.py

删除 > remove > rm

  • 删除root目录下的xx文件夹rm -rf /root/xx
  • 删除etc/yum.repos.d目录下的所有文件rm -rf /etc/yum.repos.d/*
  • 需要删除确认-i
  • 强制删除-f
  • 递归删除目录和内容-r

复制 > copy > cp

  • 复制 xxx.txt 到 /tmp目录下cp xxx.txt /tmp
  • 复制 xxx.txt 顺便改名为 xb.txtcp xxx.txt /tmp/xb.txt
  • linux下面有很多命令，一般没有办法直接处理文件夹，因此需要加上（参数 ）
    递归，复制目录以及目录的子文件cp -r
    复制文件，同时保持文件的属性不变(stat)cp -p
    相当于( -pdr)cp -a
    递归复制test文件夹，为rest2cp -r test test2
  • 输入字符串echo
    • 添加xx字符串到test.txtecho "xx" >>test.txt

链接

创建硬链接(一个文件多个路径，防止误删)ln xx xx
创建软链接(符号链接)ln -s xx xx

解压缩

tar zxvf /root/xxx.tar.gz -C /xxx/xxx			#解压文件到指定目录
tar -xJvf /root/xxx.tar.xz -C /xxx/xxx
tar zcvf xxx.tar.gz ./xxx/						#压缩当前目录下的xxx文件夹

wget

  • 用于在终端下载网络文件
    wget [参数] 下载地址

权限

d rwx r-x r-x
d文件夹/l链接/-文件 属主权限 属组权限 其他用户权限
r读/147 w写/258 x执行/369

  • test文件属主改为123chown -R 123 test

  • opt目录下的test文件夹给123管理sudo chown -R 123:123 /opt/test

  • test文件属组改为123chgrp -R 123 test

  • test最高权限(r:4,w:2,x:1)chmod -R 777 test

搜索

find / -name xxx					#查找根目录下xxx的文件
find ./ -iname xxx					#加i标识忽略大小写
cat xxx | grep 123					#匹配xx文本中的123字符
cat xxx | grep 123 -i				#加-i参数忽略大小写
find . |xargs grep -ri '123'		#查找当前目录包含文本123的文件位置

命令路径

which xx

查看

  • 查看(tac)从下到上反着看cat
  • 显示行号nl
  • 翻页查看(空格翻页，enter往下翻一行，:f查看当前行号)more
  • 与more相反(q离开)(/xx，向下查找字符串，?向上查询，n继续查找N与n方向相反)less
  • 查看文件前20行head -n 20 xx.xx
  • 与head相反tail

其他

重启关机

关闭系统前同步内存到硬盘中sync

  • 重启系统reboot systemctl reboot

  • 十分钟后重启shutdown -r nowshutdown -r +10
    #重启机器

  • 关闭系统poweroff systemctl poweroff halt

  • 马上关机shutdownshutdown -h now

  • 十分钟后关机shutdown -h 10 shutdown -h +10

  • 今天14点关机shutdown -h 14:00

用户

  • 新建用户(自动创建123用户及主目录)useradd -m 123
  • 删除用户(删除用户账号以及主目录)userdel -r 123
  • 切换用户su 123
  • 修改密码passwd 123
  • 锁定账户passwd -l 123

磁盘管理

df -h						#查看磁盘占用率
lsblk						#列出磁盘信息
lsblk -f					#列出文件系统
du -ha						#查看每个文件占用
du -hs						#查看当前文件夹大小
du -hsm /*					#检查根目录下每个文件占用的容量
du -hs /usr/src/			#查看指定目录文件占用
mount /dev/123 /mnt/123		#挂载磁盘(将外部设备123挂载到mut目录下
umount -f /mnt/123			#强制卸载
